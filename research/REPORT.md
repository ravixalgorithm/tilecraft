# Snazzymaps — What Matters

**Date:** 2026-05-23

## What it is
Free gallery + visual editor for **Google Maps JSON style arrays**. Built by Adam Krogh ~2013. 100K+ users. Ad-supported.

## Who uses it
Web designers, frontend devs, WordPress/Webflow/Framer/no-code builders. Job: make the contact-page map match the brand.

## The fact that changes everything
**Google deprecated JSON map styling on 2025-03-18.** Legacy styles migrated to Cloud Console; styling now requires a GCP project + Map ID. Snazzy's core artifact no longer drops into new code.

→ 100K users are stranded. Snazzy is still live but increasingly useless for new work.

## Competitive gap
| | Beginner-friendly | Modern format | No API key | Framer/Webflow output |
|---|---|---|---|---|
| Snazzymaps | ✓ | ✗ (dead) | ✓ | ✗ |
| Google Cloud Styling | ✗ | ✓ | ✗ | ✗ |
| Mapbox Studio | ✗ | ✓ | ✗ | ✗ |
| Maputnik | ✗ | ✓ | ~ | ✗ |
| **Tilecraft** | **✓** | **✓** | **✓** | **✓** |

No one owns *beginner-friendly + Framer/Webflow output*. That's the lane.

## Opportunity
Build a gallery + simple editor for the **MapLibre / Mapbox Style Spec** (the open standard), with one-click export to Framer and Webflow. Zero accounts, zero API keys.
